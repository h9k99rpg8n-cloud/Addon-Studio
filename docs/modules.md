# Modules

Addon Studio is built as a core app plus independent modules.

Core handles projects, files, validation, saving, settings and exports.

Alpha modules are Block Studio, Item Studio, Test Lab and Export Center.

Future modules include Texture Studio, Model Studio, Entity Studio, Animation Studio, Particle Studio, Sound Studio, Biome Studio, Structure Studio, Mission Studio, Dialog Studio, Cinematic Studio, Script Studio, Skin Studio and Shader Studio.

Rule: if one module breaks, the rest of the app should still work.
