# Roadmap

Alpha 0.1: project, manifests, blocks, items, textures, validation and export.

Alpha 0.2: JSON preview, code editor, local saving, recipes and better errors.

Alpha 0.3: Texture Studio with pixel canvas, grid, zoom and basic tools.

Alpha 0.4: Model Studio with 3D viewport and cube editing.

Beta: entities, animation, sound, particles, biomes, structures, missions, dialogs, cinematics and scripts.

Version 1.0: stable modular creator suite for Minecraft Bedrock projects.
