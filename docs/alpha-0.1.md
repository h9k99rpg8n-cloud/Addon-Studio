# Alpha 0.1 Plan

Goal: build the first working version of Addon Studio.

## Included

- Project creation
- Manifest creation
- Simple block creator
- Simple item creator
- Texture import
- Basic validation
- Export screen

## Not Included Yet

- 3D modeling
- Animation tools
- Particle tools
- Sound tools
- World tools
- Script tools

## Rule

Every button in this alpha must either work or be clearly marked as coming soon.
