# Addon Studio

Addon Studio is a modular creation studio for Minecraft Bedrock add-ons.

The goal is to help creators build complete Minecraft Bedrock experiences from one place: blocks, items, entities, textures, models, animations, sounds, particles, biomes, structures, missions, dialogs, cinematics, scripts, skins, shaders, worlds and exports.

## Vision

Addon Studio is not just a JSON editor. It is a modular suite inspired by professional Minecraft creator tools, designed to make add-on creation easier, cleaner and safer.

## Alpha 0.1 Goal

The first alpha focuses on the core workflow:

- Create a project
- Generate Behavior Pack and Resource Pack manifests
- Create simple blocks
- Create simple items
- Import PNG textures
- Generate Minecraft Bedrock JSON files
- Run basic validation in Test Lab
- Export a `.mcaddon` package

- uuid

Future modules may use:

- Three.js for 3D preview, modeling, animation and particles
- CodeMirror or Monaco Editor for JSON and scripts
- Molang tools for Minecraft Bedrock expressions
- Tauri for a future desktop version
